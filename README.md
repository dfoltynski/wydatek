#### Or run it locally

```bash
git clone https://github.com/DziadDawid/wydatek.git
cd wydatek/
cd frontend/ && npm i
npm start
```

Open new terminal

```bash
cd backend/ && npm i
nodemon app.js
```

Then you need to set your DATABASE_URI, SECRET_KEY and change .env_sample to .env
